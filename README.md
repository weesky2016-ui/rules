**纯文本 Rules 开源库**

---

**核心痛点**  
大模型不必在本地记忆臃肿的 Rules，彻底消除"记忆负担"。

**解决方案**  
通过 Gitee 直链实现**按需拉取、即取即用**。需要哪条调哪条，Agent 无需从零编写 Rules，一键引用即可运行。

**设计哲学**  
如同将 `.css` 托管在 CDN，Rules 也应该活在**云端而非本地文件**中。

**核心优势**  

| 维度 | 说明 |
|------|------|
| **灵活版本** | 标准化 Rules 文件，随时调用最新版本，告别版本碎片化 |
| **团队协同** | 一份 Rules，一个 URL，全员统一无分歧 |
| **开放自主** | 支持自由分叉（Fork），可私有化部署，完全开源，永久免费 |
